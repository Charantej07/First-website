# ☕ The Coffee Hub

This repository contains the code for "The Coffee Hub," a website showcasing a coffee shop's details, menu, hours, and contact information.

## 🌐 HTML Structure

### `index.html`

The main landing page includes:
- Navigation to different sections within the site
- A brief about Coffee Hub
- Menu details

### `hours.html`

Details the store hours and operating days.

### `contact.html`

Provides a contact form and the shop's location.

## 🎨 CSS Styling (style.css)

The CSS file defines styling, maintaining a consistent and pleasant design across the website. It includes:

- Font settings for elements like headers, paragraphs, and tables.
- Background colors for the navigation and body.
- Table formatting for menu items.

## 🛠 Usage

Clone the repository and explore each HTML file to view the Coffee Hub's various sections.
